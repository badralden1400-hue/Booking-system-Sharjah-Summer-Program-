# برنامج صيف الشارقة الرياضي - نادي خورفكان الرياضي الثقافي

تطبيق ويب مصمم بنظام Single Page Application (SPA) لإدارة عمليات تسجيل وحجز المنتسبين في البرامج والرحلات.

## الميزات الأساسية:
- **نموذج الحجز**: يتيح للمنتسبين اختيار فئة الاهتمام (برامج رياضية، رحلات ميدانية، رحلات ترفيهية) ثم اختيار النشاط المناسب بشكل ديناميكي.
- **إدارة الشعارات**: إمكانية رفع وتغيير شعار نادي خورفكان وشعار البرنامج في الصفحة الرئيسية بكل سهولة، ويتم حفظهما في متصفحك.
- **لوحة تحكم الإدارة**: 
  - عرض مؤشرات الأداء (KPIs) ورسم بياني باستخدام Chart.js
  - إضافة وحذف الأنشطة ليتمكن المنتسبون من حجزها.
  - جدول يعرض بيانات المنتسبين.
  - إمكانية تصدير بيانات المشتركين في ملف بصيغة `Excel`.

## طريقة التشغيل محلياً
بما أن التطبيق مبني بنظام HTML, CSS, JS فقط، لا تحتاج إلى أي خوادم أو برامج خاصة. 
كل ما عليك فعله هو:
1. قم بفتح ملف `index.html` في أي متصفح ويب حديث (مثل Google Chrome أو Safari أو Microsoft Edge).
2. يجب أن تكون متصلاً بالإنترنت لكي يتم تحميل خط "Tajawal" ومكتبات (Chart.js و XLSX).

## بيانات الدخول (الإدارة)
للدخول إلى بوابة "إدارة التطبيق"، يرجى استخدام البيانات التالية:

## التقنيات المستخدمة
- HTML5, CSS3, Vanilla JavaScript
- مكتبة Chart.js (لعرض الرسوم البيانية)
- مكتبة SheetJS (لتصدير الملفات إلى Excel)
- Font: Google Fonts (Tajawal)
